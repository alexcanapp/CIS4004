Description

A real-world capstone project focused on improving an existing web/mobile system. Responsibilities include fixing defects, improving reliability, and converting parts of a web repository to TypeScript for stronger type safety and long-term maintainability. Work is prioritized based on sponsor needs and sprint goals.

Skills Learned

  • Working effectively in an existing codebase (reading, tracing, and improving legacy code)
  • TypeScript migration practices (typing components, fixing compile errors, reducing unsafe casts)
  • Bug triage and reproduction strategies
  • Collaboration in Agile workflows (issue tracking, communication, iterative delivery)
  • Using developer tooling (linting, local environments, APIs)
