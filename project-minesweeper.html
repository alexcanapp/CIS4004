  • Resume

Description

A multi-screen Android app that implements classic Minesweeper gameplay. The app includes a title/instructions screen, a settings screen for configuring board size and mine density, and a game screen that renders a dynamic grid that stays centered and responsive across different grid sizes.

Skills Learned

  • Android UI layout using XML and activity navigation
  • State management for game logic (win/loss detection, reveal/flag behavior)
  • Building dynamic grids that scale based on user settings
  • Persisting user settings (e.g., using a helper class or local storage approach)
  • Debugging crashes and improving usability through iteration
